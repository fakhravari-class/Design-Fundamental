آموزش استایل‌دهی Input

Input معمولی با Text Properties

این input نمونه‌ای از font-family، font-size، font-weight، color، letter-spacing، text-transform و caret-color است.

حالت Focus و Placeholder

با کلیک روی input، حالت focus فعال می‌شود و رنگ border تغییر می‌کند و متن placeholder مخفی می‌شود.

Disabled Input

input:disabled برای زمانی استفاده می‌شود که کاربر اجازه تعامل نداشته باشد.

Read Only Input

input:read-only قابل انتخاب است اما قابل ویرایش نیست.

انواع Input

هر type رفتار مخصوص به خود را دارد اما می‌تواند استایل مشترک داشته باشد.
