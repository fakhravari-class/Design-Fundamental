# Design-Fundamental
این مخزن شامل سرفصل‌ها و تمرین‌های آموزشی (به‌صورت پوشه‌بندی‌شده) برای یادگیری **مبانی HTML و طراحی ساختار صفحات وب** است. محتوای هر جلسه/فصل داخل پوشه‌ی مربوط به همان عنوان قرار دارد. :contentReference[oaicite:1]{index=1}

---

## سرفصل‌ها
1) مبانی وب و معماری صفحات HTML  
2) مقدمه و مسیر یادگیری زبان نشانه‌گذاری HTML  
3) آشنایی با بخش Head و بهینه‌سازی متادیتا  
4) ساختاردهی اولیه سند HTML و شناخت تگ‌ها  
5) کار با متن و قالب‌بندی محتوای نوشتاری  
6) ایجاد لینک‌ها و سیستم ناوبری وب  
7) افزودن تصاویر، ویدیو و محتوای چندرسانه‌ای  
8) کار با فهرست‌ها و جداول اطلاعاتی  
9) طراحی فرم‌ها و ورودی داده  
10) ساختاردهی معنایی و طراحی منطقی محتوا  
11) عناصر تعاملی و کاربردی HTML5  
12) اصول دسترسی‌پذیری و سئو در HTML  
13) لیست تگ‌های منسوخ شده در HTML5 :contentReference[oaicite:2]{index=2}

---

## نحوه استفاده
- از بخش **Code** وارد پوشه‌ی هر جلسه شوید.
- فایل‌ها/تمرین‌های همان جلسه را مطالعه و اجرا کنید.
- پیشنهاد: بعد از هر جلسه یک فایل HTML جدید بسازید و تمرین‌های همان جلسه را بازسازی کنید.
